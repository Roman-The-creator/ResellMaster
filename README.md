# ResellMaster Inventory System
Информационная система для учета товарных запасов и аналитики продаж. Проект реализован на платформе .NET 8 с использованием архитектурных паттернов, обеспечивающих масштабируемость и тестируемость кода.

## Архитектура проекта
Приложение построено с соблюдением принципа разделения ответственности:

Слой представления (UI): Windows Forms. Взаимодействует исключительно с интерфейсами сервисов, не имея прямого доступа к базе данных.

Сервисный слой (BLL): Класс InventoryService, реализующий интерфейс IInventoryService. Содержит бизнес-логику, расчеты чистой прибыли и валидацию данных.

Слой доступа к данным (DAL): Entity Framework Core в связке с SQLite. Инкапсулирует логику хранения и извлечения сущностей через AppDbContext.


## Явное управление зависимостями
Все внешние библиотеки управляются через менеджер пакетов NuGet:
Список всех необходимых библиотек зафиксирован в файле проекта .csproj


Code-First: Схема базы данных генерируется на основе C#-классов.

Миграции: Управление изменениями структуры БД осуществляется через механизм миграций (директория /Migrations).

## Технологический стек
Framework: .NET 8.0 (Windows)

ORM: Entity Framework Core 8.0.22

Database: SQLite

Testing: xUnit
